# 出库工具

一款基于C#开发的高效文件管理工具，为办公场景量身打造。

## ✨ 主要功能

- 📊 **Excel数据导入** - 快速导入和解析Excel表格数据
- 🔍 **智能文件检索** - 根据关键词快速定位目标文件
- 📝 **批量文本替换** - 一键替换多个文件中的重复文本
- 🎯 **精准匹配** - 智能匹配算法确保替换准确性
- 💻 **友好界面** - 基于WPF的现代化用户界面
- 📋 **操作日志** - 完整记录所有操作历史
- ⚡ **高性能处理** - 支持大批量文件的高效处理

## 🛠️ 技术栈

- C# - 主要开发语言
- WPF - Windows Presentation Foundation界面框架
- .NET - 微软开发平台
- Excel - Office文档处理
