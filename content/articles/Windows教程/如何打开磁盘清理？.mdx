# 文字版

1. **打开我的电脑，鼠标右键点击C盘，选择属性**
2. **在弹出来的会话框选择磁盘清理，选择要删除的文件，点击确定即可删除**
3. **同样的步骤打开磁盘清理，点击清理系统文件**
4. **勾选要删除的文件，点击确定**
