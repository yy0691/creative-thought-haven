# PDF编辑工具

<aside>
🙌🏻 **Adobe Acrobat DC主要是用来编辑pdf，可对pdf中的图片、文字等作任意编辑；**

但对于扫描版的pdf无法识别特征，每页能识别为一整张图片，也具备识别文本功能，将图片转换为文字和图片，但使用体验不是很好。

**Enfocus PitStop 插件是Acrobat的增强工具，可以进一步修改pdf，用来去水印很强大，还有很多功能可以去探索。**

</aside>

# Enfocus PitStop 插件

### 插件安装

**下载地址：
[https://drfs.ctcontents.com/file/35786122/823208731/a8355e/Enfocus PitStop Pro.zip](https://drfs.ctcontents.com/file/35786122/823208731/a8355e/Enfocus%20PitStop%20Pro.zip)**

1. 双击**setup.exe**文件
2. 一直点击下一步即可（**一定不要更改安装路径！**）
3. 解压安装包中的**crack.rar**文件
4. 复制**crack**文件夹中的 **pitstop pro.dll** 文件
5. 粘贴该文件到安装目录：**Acrobat DC\Acrobat\plug_ins\Enfocus\PitStop Pro Resources 
   完成安装**
