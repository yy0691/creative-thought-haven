# Linux的文件系统

## 文件系统概述

windows：NTFS、FAT32    Linux：ext4、xfs

| 文件系统 | 最大文件名长度 | 最大文件大小 | 最大分区大小 |
| --- | --- | --- | --- |
| ext2 | 255b | 2TB | 16TB |
| ext3 | 255b | 2TB | 16TB |
| ext4 | 255b | 16TB | 1EB |
| XF5 | 255b | 8EB | 8EB |

### 目录说明

1. /home：用于保存用户数据，文件系统是xfs，是Linux的最新版本内核的默认文件系统
2. /boot：用于启动时加载的数据；操作系统的内核及在引导过程中使用的文件
3. /：根目录，用于创建任何目录，或挂载各个分区
4. /swap：交换分区，文件系统是swap；默认情况下和内存大小一致，用于内存和硬盘之间交换数据（虚拟内存）
5. /dev：包含了所有Linux系统所需的外部设备
6. /dev/shm：可以存储临时数据；在内存中，不在硬盘里；通常是系统内存的一般大小
7. /run：一些运行过程中跟用户或进程有关的数据，属于内核级文件
8. /sys/fs/cgroup：运行过程中CPU、内存等数据，属于内核级文件
9. /run/usr/0：编号id为0的用户 通常是root，在运行过程中产生的数据

## 常见命令

```cpp
**df -h**       //查看Linux中存在的硬盘分区情况，及每个分区的硬盘使用情况
**cd /dev
cp          //复制命令
free        //查看当前内存使用情况
cat         //查看文本文件
touch       //创建文件

shutdown -r now   //重启电脑
cd /dev/shm      //占用内存空间，通常是内内存的一半大小
ls**
```

# Linux根目录说明

- 通常情况下，可以使用/opt和/home目录，不存在敏感文件，也可以将程序安装在/opt目录下，默认安装目录：**/usr/bin或/var/**目录下
- /opt   optional，可选目录，默认是空的，交由用户使用

| / | 表示根目录，是绝对路径 |
| --- | --- |
| ./ | 表示当前目录，是相对路径 |
| ../ | 表示上一级目录，是相对路径 |
| /root | root用户的主目录 |
| /home/username | 存放普通用户的个人配置文件 |
| /bin | 存放Linux常用的命令 |
| /boot | 存放操作系统启动时要用到文件 |
| /dev |  |
| /etc | 存放系统管理时用到的配置文件和子目录 |
| /usr/sbin | 存放管理员的系统管理程序 |
| /lib | 存放系统动态链接共享库，通常后缀名是.so，类似于windows下的.dll |
| /lost+found | 系统运行异常时产生的错误，会将遗失的片段放在这里 |
| /mnt | 可临时将别的外部设备挂接在此目录下 |
| /proc | 存在系统内存中的信息 |
| /usr | 用户的应用程序和文件 |
| /tmp | 存放临时文件的目录 |

# Linux的启动级别

| 命令实例 | 作用 |
| --- | --- |
| 文件/etc/inittab | 设置默认启动级别 |
| 0 | 代表halt，关机 |
| 1 | 代表单用户模式，采用这个设置，系统只能允许一个用户登录 |
| 2 | 代表多用户模式，但不支持网络工作 |
| 3 | 代表命令行界面，即文本界面，是企业中服务器通用的启动模式 |
| 4 | 系统预留 |
| 5 | 图形界面 |
| 6 | 重启模式 |

# 常用命令

### 关机命令

```jsx
reboot  //重启
shutdown -r now //现在立即重启
shutdown -r 11:30
shutdown -r +1  //等一分钟后重启
halt   //关机
shutdown -h now
shutdown -h 11：30
init 0
init 6
```

```cpp
**top             //查看内存进程使用情况
cd ../或cd..    //切换到上一层目录（相对目录)
cd 目录名       //切换到对应目录（绝对目录，从/根目录开始）
pwd            //查看当前所在目录
du -sh         //查看文件或文件夹大小
fdisk -l       //查看磁盘分区列表
cat 文件名      //查看文本文件内容，如cat/etc/passwd可查看用户信息
cat/proc/meminfo  //查看内存信息
cat/proc/cpuinfo  //查看cpu信息**
```
