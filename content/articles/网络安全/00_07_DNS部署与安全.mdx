# 2. 域名

# 3. 监听端口

        **UDP53**

# 9. 清除DNS缓存

1. **客户机上清除缓存**

```cpp
ipconfig /flushdns
```

1. **服务器上清除缓存**

```cpp
windows服务器：DNS工具——查看——高级，调出缓存，然后右键清除缓存
```

# 10. 域名解析记录类型

```cpp
1. A记录：正向解析记录
2. CNAME记录：别名
3. PTR记录：反向解析记录
4. MX：邮件交换记录
5. NS：域名服务器解析
```
