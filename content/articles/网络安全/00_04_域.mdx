# 基本概念

1. **内网环境：**
    - 工作组：默认模式，人人平等。  
    - 域：集中管理，统一管理，权限不平等。  

2. **域的特点：**
    - 集中/统一管理。
    - 提供更高的安全性和可控性。
    
3. **域的组成：**
    - 域控制器：DC（Domain Controller），负责管理域内的所有计算机。  
    - 成员机：加入域的计算机。  

4. **域的部署：**
    - 安装域控制器——生成域环境
    - 安装了活动目录——生成域控制器 
    - 活动目录域 DC         

5. **活动目录**
    - AD
    - 特点：集中管理/统一管理

## 组策略
    - GPO（Group Policy Object）
    - 作用：通过组策略可以修改计算机的各种属性，如开始菜单、桌面背景、网络参数等。
    - 组策略在域中，是基于OU来下发的。

## 安装部署活动目录  windows server 2008
    1. 打开虚拟机，桥接到VMNET2.
    2. 配置静态IP地址.
    3. 开始——运行——输入dcpromo，安装活动目录.
    4. 勾选DNS-新林中新建域—功能级别都设置为2003—域的FQDN（.com）—设置目录服务还原密码—勾选安装后重启.
    5. 登录域，验证AD安装成功。  
    6. 验证AD是否安装成功
       - 计算机右键属性—所属域
       - DNS服务器中是否自动创建qf.com区域文件
       - 自动注册DC的域名解析记录
       - 开始—管理工具—AD用户和计算机
             comuter：普通域成员机列表
             Domain Controller：DC列表
             Users：域账号

### 域管理员组：Domain Admins

1. **组织单位OU：orgnazation union**
   
    作用：用于归类域资源（域用户、域计算机、域组）
    
2. **组策略：Group Policy =GPO**
   
    作用：通过组策略可以修改计算机的各种属性，如开始菜单、桌面背景、网络参数等
    **组策略在域中，是基于OU来下发的**
    组策略在域中下发后，用户的应用顺序是：
    
    **LSDOU：Load、站点、Domain、OU  当出现冲突时，最后一个生效，不冲突的话**
    
    打开本地组策略：gpedit.msc
