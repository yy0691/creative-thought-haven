# PKI

1. 概念
   
    名称：公钥基础设施 Public Key Infrustration
    
    作用： 通过加密技术和数字签名                                        
    
    组成
    
2. 信息安全三要素：机密性、完整性、身份验证/操作的不可否认行
3. 哪些IT领域用到PKI：1) SSL/HTTPS 2）IPsecVPN  3）部分远程访问VPN
4. 公钥加密技术
   
    作用：实现对信息加密、数字签名等安全保障
    
    加密算法：
    
    1）**对称加密算法**  加解密的密钥一致   公共密钥
                                                     **DES    3DES     AES**  
    
    ​                   
    
    2**）非对称算法**     通信双方各自产生一对公私钥；公开密钥
                                              双方各自交换公钥；
                                              公钥和私钥为互相加解密关系;
                                              公私钥不可逆推
                                                     **RSA  DH   Elgamal：基于DH**
    
    加密算法比较复杂，对于大规模数据进行加密，比较影响效率
    
    HASH算法：**MD5**   SHA （验证完整性）
    HASH值不可逆
    
    数字签名
