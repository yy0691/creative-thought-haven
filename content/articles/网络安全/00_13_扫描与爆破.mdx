## 2. Nmap- 扫描器知网

### 重要常用参数

| -sP | ping 扫描 |
| --- | --- |
| -p | 指定端口 |
| -sV | 服务版本 |
| -O | 启用操作 |
| -A | 全面扫描 |
| -oN | 保存txt |

```cpp
nmap -sP 10.1.1.1/24      //扫描10.1.1.0整个网段
nmap -p 21,23-25,3389 10.1.1.1
nmap -p 21,23 10.1.1.1 -sV      //扫描21，23的服务版本
nmap -A 10.1.1.1       //全面扫描
nmap -O 10.1.1.1       //扫描目标系统版本
nmap -sV 192.168.1.5 -oN e:\baogao.txt      //输出为txt
```
