# 优化后的文件列表和内容模板

## 文件命名规范
采用 `YYYYMMDD-标题.md` 格式，去掉日期中的连字符，标题使用小写字母，单词之间用连字符分隔。

## Front Matter 模板
```yaml
---
title: 标题
subtitle: 副标题（可选）
description: 提供完整、准确的文章摘要，长度约50-100字
author: LuoYuan
date: YYYY-MM-DD
image: 封面图片URL
link: 原文链接（如有）
category: ai-news
tags: [标签1, 标签2, 标签3]
featured: false
---
```
