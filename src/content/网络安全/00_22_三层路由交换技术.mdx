# 三层路由交换技术

1. 三层交换机= 三层路由+二层交换机
2. 三层路由引擎是可以关闭或开启的
    
    ```cpp
    **cong t
    ip routing      //开启三层路由功能
    no ip routing   //关闭三层路由功能**
    ```
    
3. 三层交换机的优点： 与单臂路由相比
    
    1）解决了网络瓶颈的问题
    
    2）解决了单点故障（虚拟接口不再依赖任何的物理接口）
    
    3）一次路由，永久交换
    
4. **三层交换机上起虚接口（配置VLAN的网关）**
    
    ```cpp
    **int vlan 10
    	ip add 10.1.1.254  255.255.255.0
    	no shut
    	exit**
    ```
    
5. **二层端口升级为三层端口**
    
    ```cpp
    **int f0/x
    	no switchport**
    ```
