# Markdown 语法指南

本指南将介绍在博客博客系统中支持的所有 Markdown 语法特性和使用方法。

## 基础语法

### 标题

使用 `#` 符号创建标题，支持 1-6 级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 文本格式

- **粗体**：`**粗体文本**`
- *斜体*：`*斜体文本*`
- ~~删除线~~：`~~删除线文本~~`
- `代码`：\`代码文本\`

### 列表

无序列表：

```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
```

有序列表：

```markdown
1. 第一项
2. 第二项
   1. 子项 2.1
   2. 子项 2.2
```

### 链接和图片

链接：`[链接文本](URL)`

图片：`![替代文本](图片URL)`

### 引用

使用 `>` 创建引用块：

```markdown
> 这是一个引用
> 可以有多行
```

### 代码块

使用三个反引号创建代码块，可以指定语言：

\`\`\`javascript
const greeting = 'Hello, World!';
console.log(greeting);
\`\`\`

## 扩展功能

### 分栏布局

使用 `columns-` 类创建多列布局：

```markdown
<div className="columns-2">

第一列内容

第二列内容

</div>
```
### 图片分栏布局
<div className="columns-2 gap-4">
  <div className="flex justify-center items-center">
    <CenteredImage width="100%" src="xxx.png" />
  </div>
  <div className="flex justify-center items-center">
    <CenteredImage width="100%" src="xxx.png" />
  </div>
</div>



### 文本高亮

使用 `highlight` 类高亮文本：

```markdown
<span className="highlight">高亮文本</span>
```

<span className="highlight">高亮文本aaaaaaaaa</span>

### 彩色背景块

使用 `highlight-` 类创建不同颜色的背景块：

```markdown
<div className="highlight-yellow">
黄色背景块
</div>

<div className="highlight-blue">
蓝色背景块
</div>

<div className="highlight-green">
绿色背景块
</div>

<div className="highlight-red">
红色背景块
</div>

<div className="highlight-purple">
紫色背景块
</div>
```
### 修改文字颜色

<span className="text-red-500">红色文字</span>
<span className="text-blue-500">蓝色文字</span>
<span className="text-green-500">绿色文字</span>

### 图片增强功能

支持设置图片宽度、高度和缩放：

```markdown
![图片描述](图片URL){width=500 height=300 scale=0.8}
```

## 最佳实践

1. **文档结构**
   - 使用适当的标题层级
   - 保持内容层次清晰
   - 适当使用空行分隔段落

2. **图片使用**
   - 提供有意义的替代文本
   - 合理设置图片尺寸
   - 优先使用 WebP 格式

3. **代码展示**
   - 指定正确的语言以获得语法高亮
   - 保持代码格式整洁
   - 添加必要的注释

4. **样式使用**
   - 合理使用高亮和背景色
   - 避免过度使用装饰性样式
   - 确保视觉层次分明
