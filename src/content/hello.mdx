# 欢迎来到我的博客

这是一个使用MDX创建的示例文章。MDX允许我们在Markdown中使用React组件，这使得我们的博客内容更加丰富和交互。
这是行内代码示例：console.log('Hello World')

## 特性

- 支持所有标准的Markdown语法
- 可以导入和使用React组件
- 支持代码高亮

```javascript
const greeting = 'Hello, World!';
console.log(greeting);
```
