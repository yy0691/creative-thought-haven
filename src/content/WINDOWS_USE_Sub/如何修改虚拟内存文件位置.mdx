# 文字版

1. 鼠标右键点击此电脑，点击属性
2. 弹出的会话框往下拉，找到高级系统设置
3. 在性能框中点击设置
4. 弹出来的会话框点击高级，点击更改
5. 取消勾选自动管理……，选择其他盘，点击设置，点击确定
